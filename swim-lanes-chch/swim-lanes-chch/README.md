# 🌊 Swim Lanes Ōtautahi

Find the quietest lane swimming across all Christchurch Recreation & Sport pools.

## Deploy to Vercel

1. **On GitHub** — create a new repo called `swim-lanes-chch` (already done ✓)

2. **Upload these files** — go to your GitHub repo, click "uploading an existing file", drag the contents of this zip in

3. **On Vercel** — import the repo, select **Vite** as framework, click Deploy

URL will be: `https://swim-lanes-chch.vercel.app`

## Features
- All 7 CCC pools ranked by lane quietness
- Filter by pool, day (full year), time of day
- Update button fetches live data from CCC
- Tap any pool for 30-min slot breakdown
- Works as iPhone home screen app

## Phone 📱
Open URL in Safari → Share → Add to Home Screen
